## 🎯 **Overview**  
**Born2BeRoot** is a project at **[1337 Coding School](https://1337.ma)** (part of the **42 Network**) where you set up a **virtual machine (VM)** running **Debian** or **Ubuntu** and configure it to meet specific security and system administration requirements. This project teaches you the fundamentals of **server administration**, **system security**, and **Linux operating systems**. It’s a great way to understand how to manage and secure a server.

---

## 🧠 **What You'll Learn**  
- **📖 Virtualization**:  
  Learn how to set up and manage a virtual machine using **VirtualBox** or **VMware**.  
- **🧩 Linux System Administration**:  
  Configure users, groups, and permissions on a Linux system.  
- **⚙️ System Security**:  
  Implement security measures like **firewalls**, **SSH hardening**, and **password policies**.  
- **🧠 Scripting and Automation**:  
  Write **bash scripts** to automate system monitoring and maintenance tasks.  

---

## 🚀 **Features**  
- **✅ Core Functionality**:  
  - Set up a **Debian** or **Ubuntu** server on a virtual machine.  
  - Configure **users**, **groups**, and **sudo permissions**.  
  - Implement a **strong password policy** and **user management system**.  

- **✨ Advanced Features**:  
  - Set up a **firewall** using `ufw` (Uncomplicated Firewall).  
  - Harden **SSH** access (e.g., disable root login, change default port).  
  - Configure **cron jobs** to automate system monitoring tasks.  
  - Write a **monitoring script** to display system information (e.g., CPU, memory, disk usage).  

- **📦 Organized Setup**:  
  Clean, well-documented configuration files and scripts for easy maintenance.

## 🛠️ **Technologies Used**  
- **💻 Debian/Ubuntu**: The Linux distribution used for the server.  
- **📜 VirtualBox/VMware**: Virtualization software to create and manage the VM.  
- **🧠 Bash Scripting**: For automation and monitoring tasks.  
- **⚙️ UFW (Uncomplicated Firewall)**: For configuring the firewall.  
- **🧠 SSH**: For secure remote access to the server.
